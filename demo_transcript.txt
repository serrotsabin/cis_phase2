SUMMARY OF DEMONSTRATED FEATURES

Multi-client support (3 simultaneous clients)
Real-time output broadcasting to all clients
Floor control protocol (FIFO queue)
Control request (Ctrl+T)
Control release (Ctrl+R)
User list (Ctrl+L)
Auto-transfer on controller disconnect
Clean disconnect (Ctrl+X)
Proper process cleanup
